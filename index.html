部署指南
3 种方式，5 分钟快速部署到 Cloudflare Workers
1

Wrangler CLI

使用 Cloudflare 官方 CLI 工具

npm install -g wrangler
wrangler login
wrangler deploy --main _workers.js
2

Dashboard

通过 Cloudflare 网页控制台

登录 Cloudflare Dashboard →
Workers & Pages → 创建 Worker
粘贴 _workers.js → 保存部署
3

一键部署

点击 GitHub 仓库中的部署按钮

点击 "Deploy to Cloudflare Workers"
授权 GitHub 和 Cloudflare
自动部署完成

常见问题
解答您可能遇到的疑问
部署后出现 404 错误？
确保复制的是 _workers.js 的完整内容，而非 index.html。
限流触发后如何解除？
限流周期为 60 秒，等待 60 秒后自动解除，响应头包含 retry-after: 60。
WebSocket 连接失败？
确保客户端支持 WebSocket 协议，检查网络环境是否允许 WebSocket 连接。
带宽测速结果不准确？
测速受网络波动影响，建议多次测试取平均值，有线网络更稳定。
CPU 测试结果差异大？
Cloudflare Workers CPU 资源共享，结果有波动，建议多次测试取平均。
支持 IPv6 吗？
完全支持，Worker 会自动检测并显示 IPv4 和 IPv6 地址。
