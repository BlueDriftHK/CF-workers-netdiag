零基础部署 · 两种方式任选

免费套餐每日10万次请求，个人使用绰绰有余

🌐 方式一：Cloudflare 控制台 (Web)

  1. 登录 Cloudflare Dashboard → Workers & Pages
  2. 点击「创建应用程序」→「创建 Worker」
  3. 将仓库中 _worker.js 代码粘贴至编辑器
  4. 「保存并部署」→ 获得 *.workers.dev 专属域名
获取最新 worker 脚本 →

🛠️ 方式二：Wrangler CLI (开发者)

git clone https://github.com/ASIACOMKHK/CF-workers-ipcheck.git
cd CF-workers-ipcheck
npx wrangler login
npx wrangler deploy

一键部署，版本管理更便捷

开发者 API · 嵌入式集成

除了可视化界面，Worker 提供丰富的 REST 风格接口，适合自动化监控与二次开发。

接口端点参数返回说明
/speedtest?size=1048576 (字节)随机二进制数据，用于带宽测试
/cpu-test?n=500000执行浮点运算，返回耗时和结果
/doh-compare无对比 Cloudflare / Google DoH 解析时延(ms)
/wasm-test无运行 WebAssembly 斐波那契基准，返回执行时间
/websocket-bench可选WebSocket 双向通信延迟测试
/concurrency?requests=20模拟并发请求，测试吞吐性能
示例：curl https://你的worker域名/speedtest?size=5242880 下载5MB测速文件

📊 可视化实时图表

前端集成ECharts动态展示延时曲线和历史抖动，网络健康状况一览无余，并提供速度仪表盘。

🔐 隐私优先 · 透明化安全评估

展示TLS版本/密码套件、ECH状态以及Cloudflare Bot评分，快速判断网络环境是否存在中间人风险。
