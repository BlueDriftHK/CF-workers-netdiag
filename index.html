三分钟快速部署

无需本地环境，无需安装依赖，复制粘贴即可上线

创建 Cloudflare Worker

登录 Cloudflare 控制台 → Workers & Pages → Create application → Create Worker，输入名称后点击 Deploy。

替换完整源码

点击 Quick edit 进入编辑器，删除默认全部代码，粘贴 _workers.js 完整源码。

保存并访问

点击 Save and deploy，等待部署完成。访问分配的 *.workers.dev 域名即可打开诊断面板。

（可选）绑定 KV 命名空间

创建 KV 命名空间，在 Worker 设置中绑定变量名为 SPEED_HISTORY，即可启用测速历史与用量统计功能。

// ============================================================
// NetSight Pro - Apple 极简网络诊断工具
// Cloudflare Worker 完整优化版 | v4.0
// ============================================================

// CORS & 安全响应头
const CORS_HEADERS = {
  'access-control-allow-origin': '*',
  'access-control-allow-methods': 'GET, HEAD, OPTIONS',
  'cache-control': 'no-store'
};

const SECURITY_HEADERS = {
  'x-content-type-options': 'nosniff',
  'x-frame-options': 'DENY',
  'x-xss-protection': '1; mode=block',
  'strict-transport-security': 'max-age=31536000'
};

// 内置限流机制
const rateLimit = new Map();
function isRateLimited(ip, maxRequests = 60, windowMs = 60000) {
  const now = Date.now();
  const requests = rateLimit.get(ip) || [];
  const recent = requests.filter(t => now - t < windowMs);
  if (recent.length >= maxRequests) return true;
  recent.push(now);
  rateLimit.set(ip, recent);
  return false;
}

// 主入口
addEventListener('fetch', event => {
  event.respondWith(handleRequest(event));
});

async function handleRequest(event) {
  const url = new URL(event.request.url);
  const clientIp = request.headers.get('cf-connecting-ip');
  
  // 限流拦截
  if (isRateLimited(clientIp)) {
    return new Response(JSON.stringify({ error: 'Too Many Requests' }), { status: 429 });
  }
  
  // 路由分发
  switch(url.pathname) {
    case '/':           return renderMainPage();
    case '/health':     return handleHealth();
    case '/speedtest':  return handleSpeedTest(url);
    case '/cpu-test':   return handleCpuTest(event);
    case '/ws-test':    return handleWebSocket(event);
    case '/http2-test': return handleHttp2Test(event);
    // ... 更多接口实现
  }
}

// IP 欺诈风险评分
async function getIpFraudScore(ip) {
  try {
    const resp = await fetch(`https://scamalytics.com/api/ip/${ip}`);
    const data = await resp.json();
    return {
      score: data.score,
      riskLevel: data.score >= 80 ? 'high' : 'low'
    };
  } catch (e) {
    return null; // 超时降级，不影响主功能
  }
}

常见问题

关于部署、使用与限制的常见疑问解答

部署这个工具需要付费吗？

完全免费。Cloudflare Workers 免费套餐每天提供 10 万次请求额度，个人使用完全足够。 KV 存储也有免费额度，测速历史等功能同样可以免费使用。如需更高配额可按需升级付费套餐。

为什么 Worker 不能做 TCP ping / traceroute？

Cloudflare Workers 运行在 JavaScript 沙箱中，仅支持 HTTP / WebSocket 协议，无法发送原始 ICMP 或 TCP 数据包。 本项目通过 HTTP 请求耗时来模拟延迟测试（HTTP Ping），结果可近似反映网络质量。如需真正的 MTR 路由追踪，需要配合支持 TCP 的边缘计算服务。
