部署指南
3 种方式，5 分钟快速部署到 Cloudflare Workers
1

Wrangler CLI

npm install -g wrangler
wrangler login
wrangler deploy --main _workers.js

2

Dashboard 部署

登录 Cloudflare Dashboard → Workers & Pages → 创建 Worker → 粘贴 _workers.js 代码 → 保存部署

3

一键部署

点击 GitHub 仓库中的 "Deploy to Cloudflare Workers" 按钮，自动完成部署

前置条件

Cloudflare 账号 + 域名（可选，可使用 workers.dev 子域名）+ Node.js 18+

注意

部署时请使用 _workers.js 文件（以下划线开头），而非 index.html
