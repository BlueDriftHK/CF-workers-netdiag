项目介绍

部署在 Cloudflare Workers 边缘节点的专业级网络诊断工具

NetSight Pro 是一款功能强大的网络诊断工具，利用 Cloudflare Workers 的边缘计算能力， 提供实时网络监控、多维度性能测试和全面的安全协议检测。极光主题的现代化界面， 为企业级网络诊断提供了优雅的解决方案。

功能特性

全面的网络诊断工具集

实时延迟监控

每2秒自动测量RTT，实时趋势图表，连接质量评估

带宽测速

多档位测试（100KB/500KB/2MB），精确计算传输速率

命令行测试示例

# 健康检查
curl "https://your-worker.dev/health"

# 带宽测试（100KB）
curl "https://your-worker.dev/speedtest?size=102400"

# CPU 性能测试
curl "https://your-worker.dev/cpu-test?n=500000"

# HTTP/2 检测
curl "https://your-worker.dev/http2-test"

# 并发测试
curl "https://your-worker.dev/concurrent-test?count=4&size=2048"

部署指南

快速部署到 Cloudflare Workers

方式一：Wrangler CLI 部署

# 安装 Wrangler CLI
npm install -g wrangler

# 登录 Cloudflare
wrangler login

# 部署
wrangler deploy

方式二：Cloudflare Dashboard 部署

1. 登录 Cloudflare Dashboard
2. 进入 Workers & Pages > 创建应用程序
3. 创建 Worker 并粘贴代码
4. 保存并部署

wrangler.toml 配置示例

name = "netsight-pro"
main = "worker.js"
compatibility_date = "2024-12-01"

# 可选：自定义域名
# routes = [
#   { pattern = "your-domain.com", custom_domain = true }
# ]
