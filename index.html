一键部署到 Cloudflare Workers

# 1. 安装 Wrangler CLI
npm install -g wrangler

# 2. 登录 Cloudflare
wrangler login

# 3. 创建 wrangler.toml
name = "netsight-pro"
main = "worker.js"
compatibility_date = "2024-12-01"

# 4. 部署
wrangler deploy
1

复制 worker.js 代码

将完整的 NetSight Pro Worker 代码保存为 worker.js 文件

2

配置 wrangler.toml

创建配置文件，设置 Worker 名称和兼容性日期

3

部署到 Cloudflare

运行 wrangler deploy 命令，自动分配 workers.dev 域名
