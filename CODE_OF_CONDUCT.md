# 贡献者行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，作为贡献者和维护者，我们承诺：无论年龄、体型、残疾、种族、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、宗教或性取向如何，参与我们的项目和社区都不会受到骚扰。

我们承诺以建设性、尊重和专业的方式参与项目，为项目和社区做出积极贡献。

---

## 我们的标准

### 积极的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

### 不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如物理地址或电子邮件地址）
- 在专业场合中可能被合理认为不适当的其他行为

---

## 我们的责任

项目维护者负责明确可接受行为的标准，并对任何不可接受的行为采取适当、公平的纠正措施。

项目维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，同时可以暂时或永久封禁任何他们认为不当、威胁、冒犯或有害的行为者。

---

## 适用范围

本行为准则适用于项目所有空间（包括 GitHub 仓库、Issue、Pull Request、Discussions 等），也适用于个人在公共场合代表项目或其社区时。代表项目的行为包括使用官方项目电子邮件地址、通过官方社交媒体账号发布内容，或担任线上或线下活动的指定代表。项目代表的范围可由项目维护者进一步定义和明确。

---

## 执行

如果您发现任何违反本行为准则的行为，请通过以下方式向项目维护者报告：

- **电子邮件**：`asiacomk@gmail.com`
- **GitHub Issues**：您也可以私下联系维护者（通过 GitHub 私信，如果可用）

所有投诉都将被审查和调查，并根据具体情况做出相应且必要的回应。项目维护者有义务对事件报告者保密。

如果维护者认为必要，可以在不事先通知的情况下采取临时或永久的处理措施，包括警告、暂时封禁或永久封禁。
